**Keywords:** proposal, scope-of-work, ROI, value-based-pricing, case-study, social-proof, closing, B2B, SMB

---

## Instructions for Use

This template is completed by the Compass Strategic Advisor agent using discovery call notes loaded into context. The agent fills in all [BRACKETED] fields. The completed proposal is reviewed by the CorCompass operator before sending.

All proposals must be sent within 48 hours of the discovery call. Proposals sent after 72 hours have a significantly lower close rate.

## Compass Agent Completion Instructions

When completing this template, the Compass Strategic Advisor agent must:

1. Replace all [BRACKETED] fields with specific, accurate information from the discovery call notes.
2. Ensure the ROI calculation in Section 1 uses the exact numbers from the discovery call (do not estimate; if numbers are missing, flag for operator review before sending).
3. Select the case study in Section 4 that most closely matches the prospect's vertical and problem type.
4. Set the proposal valid date to exactly 14 days from today.
5. Flag any fields that could not be completed due to missing information from the discovery call.
6. Output the completed proposal as a clean Markdown document for operator review.
